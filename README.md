# TicTacToe
#### Video Demo:  https://youtu.be/PvLVknIvQIk
#### Description:


This is a simple implementation of the Tic-Tac-Toe game using Pygame. The game allows the user to play against an AI player.

### Requirements
To run the game, you need to have the following installed:

Pygame

### How to Play
Run the game by running the main.py file.

Choose a player by clicking on the "Play as X" or "Play as O" button.

The game board will be displayed, and the player's turn will be indicated at the top of the screen.

To make a move, simply click on an empty square on the board.

If playing against the AI, the computer will make its move automatically.

The game ends when a player wins or when there is a tie. To play again, click on the "Play Again" button at the bottom of the screen.
